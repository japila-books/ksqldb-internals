# Formats

ksqlDB comes with the following built-in [Format](Format.md)s for keys and values:

Format Name | Format
------------|-------
 AVRO | `AvroFormat`
 DELIMITED | `DelimitedFormat`
 JSON | `JsonFormat`
 JSON_SR | `JsonSchemaFormat`
 KAFKA | [KafkaFormat](KafkaFormat.md)
 NONE | `NoneFormat`
 PROTOBUF | `ProtobufFormat`
